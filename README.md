## Getting Started

### View locally

```bash
# Clone the repository
git clone https://github.com/your-username/portfolio.git

# Open in browser — no server required
open index.html
```

### Serve locally (optional)

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

Then visit `http://localhost:8080`.

## Customisation

### Update contact details
Search for `hello@shaneramirez.com` and your LinkedIn URL in `index.html` and replace with updated info.

### Add a new work sample
In the **Work Gallery** section, duplicate an existing portfolio card block and update:
- The file name and type badge
- The description text
- The `href` on the download link pointing to your new file in `/assets/files/`

### Add a new certification
In the **Certifications** section, add a new credential block with:
- Issuer name
- Issue date and credential ID
- Link to the verified credential URL

### Update stats (hero section)
The four stat counters (Leads Managed, Years in Real Estate, Hours Saved, Tools Mastered) are hardcoded — update the numbers directly in the HTML when your numbers grow.
